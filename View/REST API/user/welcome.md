# Welcome Page API

API for Welcome Page.

Url : `/MyEng/Welcome`

--------
## Direct to SignUp

**Describle** : When click button "Sign Up" in Interface 

**URL** : `/MyEng/SignUp`

**Method** : `GET`

### **Success Response**

**Code** : `200 OK`

**Content** : Return File 'SignUp'

--------
## Direct to SignIn

**Describle** : When click button "Sign In" in Interface 

**URL** : `/MyEng/SignIn`

**Method** : `GET`

### **Success Response**

**Code** : `200 OK`

**Content** : Return File 'SignIn'
